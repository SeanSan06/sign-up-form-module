# Sign-up form module
A simple front-end UI sign-up form that can be used in other projects.  
[https://seansan06.github.io/sign-up-form-module/](https://seansan06.github.io/sign-up-form-module/)

# How to use
Simply pull down the code and replace the HTML with what you want to integrate it into your website.

# How its made
Technologies used: HTML, CSS, and JavaScript.

HTML: Make the contents of the website. Utilize HTML form tag and built in validation checkers.

CSS: Basic CSS to style the website.

JavaScript: Used JavaScript to handle when form submission events happen.

Github pages: Used to host the landing page.

# What I learned
I learned how to use HTML and JavaScript to perform form validation. I learned how apply the right parameters to the tags so that the backend can easily pick up on the form inputs.
